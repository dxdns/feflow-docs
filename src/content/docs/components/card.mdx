A flexible container component used to group related content with optional styling and padding.

## Usage

<Tabs>
  <TabItem label="Demo">
    <Card variant="contained">
      <h2>test card</h2>
    </Card>
  </TabItem>
  <TabItem label="Code">
    ```svelte
    <script lang="ts">
      import { Card } from "@dxdns/feflow"
    </script>

    <Card>
      <h2>test card</h2>
    </Card>
    ```
  </TabItem>
</Tabs>

<br />
<br />

| Prop  | Type     | Default | Description                          |
|-------|----------|---------|--------------------------------------|
| none  | –        | –       | This component accepts no props.     |

| Slot     | Description                         |
|----------|-------------------------------------|
| default  | Content to be placed inside the card. |

# Accessibility

The card is rendered as a `<div>` and inherits all standard HTML accessibility behavior. You can extend it with semantic elements inside (like `<article>`, `<section>`, or `<h2>`).

# Best Practices

- Use `<Card>` to visually group related UI elements.
- Place semantic content inside (e.g., headings, paragraphs).
- Combine with utility classes or wrapper components for spacing and layout.
